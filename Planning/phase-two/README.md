# phase-two/

Backlog folder for the Mira Palace project.

- `FUTURE_ENHANCEMENTS.md` — every idea that isn't in the current phase. Format per `Planning/AGENT_WORKING_AGREEMENT.md §7.1`: **what / why / when / effort (S/M/L) / who raised it**. Nothing is deleted — closed items move to the bottom under "Closed / decided" with the date and reason.

Promoting an item into a real phase: take it out of this file, add it to `MEMORY.md`'s phase table, scope it, then build. Do not start work on a backlog item without promoting it first.
