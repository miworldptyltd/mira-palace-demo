# claude-memory/

This folder is the agent's persistent state for the Mira Palace project.

- `MEMORY.md` — live state, phase table, what's built, changelog. Updated every session or after any meaningful change. Per the Agent Working Agreement §1.1 the agent reads this file at the start of every session.
- `README.md` — this file.

**Do not delete** entries from `MEMORY.md`. Superseded facts move to the dated changelog at the top; current state stays current. See `Planning/AGENT_WORKING_AGREEMENT.md §1` for the full rule.
